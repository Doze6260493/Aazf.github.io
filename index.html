Visual Studio Code

¿Que es Visual Studio Code?

Visual Studio Code es un editor de código fuente que permite trabajar con diversos lenguajes de programación, admite gestionar tus propios atajos de teclado y refactorizar el código.Fue lanzado el 29 de abril de 2015 y lo tienes disponible de forma gratuita en LINK PARA DESCARGAR VISUAL STUDIO CODE


Notepad++

¿Que es Notepad++?

Notepad++ es un editor de texto y de código fuente libre con soporte para varios lenguajes de programación. Con soporte nativo para Microsoft Windows. Se parece al Bloc de notas en cuanto al hecho de que puede editar texto sin formato y de forma simple.Fecha del lanzamiento inicial: 24 de noviembre de 2003 y lo tienes disponible de forma gratuita LINK PARA DESCARGAR NOTEPAD++


Sublime Text

¿Que es Sublime Text?

Sublime Text es un editor de texto y editor de código fuente. Está escrito en C++ y Python para los plugins.​ Desarrollado originalmente como una extensión de Vim, con el tiempo fue creando una identidad propia. Aún conserva un modo de edición tipo vi llamado Vintage mode.Fecha del lanzamiento inicial: 18 de enero de 2008 y lo tienes disponible de forma gratuita en LINK PARA DESCARGAR SUBLIME TEXT ES LA ESTRUCTURA DE UNA PAGINA WEB ES LA BASE DE TODO NECESITAREMOS HTML

Python

¿Que es Python?

Python es un lenguaje de programación interpretado cuya filosofía hace hincapié en la legibilidad de su código. Se trata de un lenguaje de programación multiparadigma, ya que soporta parcialmente la orientación a objetos, programación imperativa y, en menor medida, programación funcional. Es un lenguaje interpretado, dinámico y multiplataforma y lo tienes disponible de forma gratuita en LINK PARA DESCARGAR PYTHON

Java

¿Que es Java

Java es un lenguaje de programación utilizado para crear software compatible con una gran diversidad de sistemas operativos. Este lenguaje tiene la particularidad de ser compilado e interpretado al mismo tiempo; esto significa que es un lenguaje simplificado que convierte automáticamente el código en instrucciones de máquinay lo tienes disponible de forma gratuita en LINK PARA DESCARGAR JAVA

Código Básico HTML

Este codigo es super importante sera el corazon de la programacion para ti asi que te recomendamos que lo tengas en mente para que memorizar 

  <!DOCTYPE html>
  <html lang="en">
  <head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Mi Página</title>
  </head>
  <body>
  <h1>¡Hola, mundo!</h1>
  </body>
  </html>

Ese es el codigo basico de html cabe agregar que se le agrego la variable h1 para probar la funcionalidad del codigo

Código HTML con Imagen

Con el codigo basico html puedes hacer muchas cosas. Te enseñare las mas basicas

Como colocar una imagen

Código HTML con Imagen y Botón de Copia 
                <!DOCTYPE html>
                <html lang="es">
                <head>
                    <meta charset="UTF-8">
                    <meta name="viewport" content="width=device-width, initial-scale=1.0">
                    <title>Código HTML con Imagen y Botón de Copia</title>
                </head>
                <body>
                    <h1>Mi Página con Código HTML y una Imagen</h1>
                    <img src="ruta_de_la_imagen.jpg" alt="Descripción de la imagen">
                </body>
                </html>

Como colocar un parrafo

                        <meta charset="UTF-8">

Código HTML con Enlace y Botón de Copia

Como crear un Enlace 

            <!DOCTYPE html>
            <html lang="es">
            <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <title>Código HTML con Enlace </title>
            </head>
            <body>
                <h1>Mi Página con Código HTML y un enlace</h1>

            </body>
            </html>
Código HTML con Tabla y Botón de Copia

Como crear una tabla

            <!DOCTYPE html>
            <html lang="es">
            <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <title>Código HTML con Tabla y Botón de Copia</title>
            </head>
            <body>
                <h1>Mi Página con Código HTML y una Tabla</h1>
                <table class="page-table">
                    <thead>
                        <tr>
                            <th>Encabezado 1</th>
                            <th>Encabezado 2</th>
                        </tr>
                    </thead>
                    <tbody>
                        <tr>
                            <td>Celda 1</td>
                            <td>Celda 2</td>
                        </tr>
                        <tr>
                            <td>Celda 3</td>
                            <td>Celda 4</td>
                        </tr>
                    </tbody>
                </table>
            </body>
            </html>
Código HTML con Encabezado y Botón de Copia

Como crear encabezados

            <html lang="es">

                <meta charset="UTF-8">

                <h2 class="page-subheader">Este es mi encabezado h2</h2>

                <h4 class="page-subsubheader">Este es mi encabezado h4</h4>
                <p>Mi Página con Código HTML y encabezados</p>
            </body>
            </html>
